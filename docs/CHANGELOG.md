## [v0.1.0] - 2025-12-23

因为兴趣写的自用小工具，主要把文件搜索和剪贴板记录凑在了一起。

**主要功能：**

* **文件搜索**：支持全盘搜索，可以切换正则匹配或区分大小写，搜到后支持一键打开文件所在位置。
* **剪贴板回溯**：自动记录复制过的文字、图片和多个文件路径，点击历史记录即可重新写回剪贴板。
* **外观设置**：适配 Win11 风格，支持深色模式跟随系统，并可以最小化到系统托盘运行。

没什么复杂的，纯粹为了自己用着方便。
