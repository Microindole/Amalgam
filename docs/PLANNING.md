### 1. ⚡️ 核心体验增强 (High Priority)

这些功能是效率工具的“基石”，能极大提升可用性。

* **全局快捷键 (Global Hotkey)**
* **现状**：目前只能通过点击托盘图标或任务栏唤起窗口。
* **建议**：注册全局快捷键（如 `Alt + Space` 或 `Ctrl + Shift + V`）来显示/隐藏窗口。
* **技术栈**：使用 Tauri 的 `GlobalShortcut` 插件。
* **价值**：无需鼠标，想用即呼，用完即走。


* **键盘导航 (Keyboard Navigation)**
* **现状**：目前列表操作依赖鼠标点击（点击复制、点击展开）。
* **建议**：支持 `↑` `↓` 键选择列表项，`Enter` 键复制/打开，`Esc` 隐藏窗口。
* **价值**：真正的效率工具应该允许用户双手不离键盘。



### 2. 🔍 Trace (剪贴板) 进阶

让剪贴板管理更智能。

* **历史记录搜索 (Search History)**
* **建议**：在 Trace 页面顶部增加一个搜索框（或者复用现有的搜索逻辑），允许用户输入关键词快速过滤那 50 条历史记录。
* **场景**：你记得半小时前复制过一段代码，但不想翻列表，直接搜代码里的变量名。


* **收藏/置顶 (Pin/Star)**
* **建议**：允许用户“钉住”某些常用的片段（如身份证号、常用命令），使其不会被新记录顶掉。


* **智能过滤 (Smart Filter)**
* **建议**：添加分类标签按钮：[全部] [图片] [链接] [代码]。
* **技术**：可以在 Rust 端或前端通过正则简单判断内容类型（URL、Email、颜色代码等）。



### 3. 📂 Seek (文件检索) 进阶

让文件搜索不仅是“找文件”，而是“预览文件”。

* **文件快速预览 (Quick Look)**
* **建议**：选中文件后，按 `Space` 键（类似 macOS）在右侧或弹窗预览文件内容。
* **实现**：
* **文本/代码**：直接读取前 1KB 内容显示。
* **Markdown**：简单渲染。
* **图片**：显示缩略图。


* **价值**：不用打开笨重的 IDE 或 Word，就能确认是不是你要找的文件。


* **更多操作上下文 (Context Menu)**
* **建议**：右键点击搜索结果，提供更多选项：
* “复制文件路径”
* “复制文件内容” (无需打开文件)
* “在终端中打开”
